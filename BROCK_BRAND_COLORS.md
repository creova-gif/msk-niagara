## Usage Guidelines

### Primary Actions
Use **Brock Blue (#1C2D5D)** for:
- Primary buttons
- Main navigation
- Headings
- Important text

### Secondary Actions & Accents
Use **Brock Red (#F20014)** for:
- Secondary buttons
- Call-to-action elements
- Links
- Hover states
- Highlights

### Neutral Elements
Use **Brock Grey (#C8C9CB)** for:
- Borders
- Disabled states
- Secondary text
- Subtle backgrounds
